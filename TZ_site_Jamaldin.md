## 0. Суть за 30 секунд

Одностраничный лендинг (one-page, скролл вниз) для личного бренда **Джамалдин**. Продаёт три услуги:
1. Создание сайтов;
2. Сценарии для Reels;
3. Монтаж Reels.

Задача сайта — с первого экрана вызвать доверие и ощущение премиальности, объяснить бизнесу зачем ему это, и подвести к заявке. Аудитория — предприниматели и эксперты **любой ниши** (услуги, товары, консалтинг, локальный бизнес, онлайн-школы, медицина, строительство и т.д.).

Ключевая эмоция: **«вау, это дорого и профессионально, я хочу так же»**.

## 1. Позиционирование и тон

- **Позиционирование:** не «фрилансер», а цифровой специалист, который делает бизнесу премиальное цифровое лицо и системный контент.
- **Обещание:** клиента выбирают ещё до первого контакта — потому что сайт и контент выглядят дороже конкурентов.
- **Тон копирайта:** уверенный, спокойный, без воды и без крика. Никаких «!!!», никаких «лучший в мире». Премиум говорит тихо.
- **Универсальность:** везде подчёркиваем, что услуги подходят под любую отрасль. Формулировки без привязки к конкретной нише.

## 2. Дизайн-система (Design System reference)

### 2.1 Концепция

**«Цифровое зрение» (Digital Vision).** Референс — макро-снимок кибер-глаза, в радужке которого отражаются потоки данных и бинарный код (синее свечение, тёмный фон, тёплые боке-блики на периферии).

Идея, которую переносим в дизайн: *технология, которая видит и понимает бизнес*. Тёмное «космическое» пространство + холодное синее свечение как основной акцент + редкие тёплые янтарные блики как «человеческое тепло» внутри технологичного мира.

**Сигнатурный элемент (то, что запомнят):** живая «цифровая радужка» / апертура в hero — концентрические кольца с бегущими цифрами и мягким свечением, которые реагируют на курсор. Это прямая отсылка к референсу и главный вау-акцент. Он должен быть единственным «громким» элементом — всё остальное держим тихо и дисциплинированно.

### 2.2 Цветовая палитра (токены)

```
--bg-void:        #05070F   /* глубокий сине-чёрный фон, основа */
--bg-surface:     #0A0F1D   /* приподнятые поверхности, секции */
--bg-glass:       rgba(14, 22, 42, 0.55)  /* стеклянные карточки */

--accent-cyan:    #00D9FF   /* основной неон, свечение, ключевые акценты */
--accent-blue:    #2E7BFF   /* глубокий синий для градиентов и кнопок */
--accent-amber:   #FF9D2E   /* тёплый боке-блик, ТОЛЬКО точечно (1–2 места) */

--text-primary:   #EAF2FF   /* заголовки и основной текст */
--text-secondary: #8598B8   /* подписи, вторичный текст */
--text-muted:     #4E5D78   /* дисклеймеры, мелочь */

--stroke:         rgba(0, 217, 255, 0.14)   /* тонкие границы */
--glow:           rgba(0, 217, 255, 0.35)   /* тень-свечение */
```

Правила использования:
- Фон всегда тёмный (`--bg-void` / `--bg-surface`).
- Синий (cyan → blue) — основной акцент и градиенты.
- **Янтарный использовать крайне сдержанно** — максимум в 1–2 точках на всём сайте (например, микро-блик у ключевой цифры или подсветка активного шага). Он «дорогой» только пока редкий.
- Градиент-подпись: линейный `--accent-cyan → --accent-blue` под углом 135°.

### 2.3 Типографика

- **Display / заголовки:** `Space Grotesk` (веса 500–700). Технологичный, но не «геймерский». Крупно, с плотным трекингом на H1.
- **Body / текст:** `Inter` (400–500). Чистая читаемость.
- **Утилитарный / цифры, лейблы, счётчики:** `JetBrains Mono` — для цен, цифр статистики, тех-подписей (усиливает «дата»-эстетику из референса).

Тип-шкала (desktop):
```
H1  clamp(44px, 6vw, 88px)   / line-height 1.02 / letter-spacing -0.02em / weight 700
H2  clamp(30px, 4vw, 52px)   / line-height 1.08 / weight 600
H3  22–26px / weight 600
Body-lg 18–20px / line-height 1.6
Body    16px  / line-height 1.65
Label / mono  13px / uppercase / letter-spacing 0.14em
```

### 2.4 Сетка, отступы, форма

- Контейнер: max-width **1200px**, боковые поля 24px (mobile) / 64px (desktop).
- Вертикальный ритм секций: 120–160px между секциями (desktop), 72–88px (mobile).
- Скругления: карточки `16px`, кнопки `12px`, крупные блоки `24px`.
- Тонкие границы `1px solid --stroke` вместо жирных линий.
- Базовая сетка 8px.

### 2.5 Эффекты и стиль поверхностей

- **Glassmorphism:** карточки — полупрозрачный `--bg-glass`, `backdrop-filter: blur(20px)`, граница `--stroke`, лёгкое внутреннее свечение.
- **Свечение (glow):** ключевые элементы (кнопка, радужка, активная карточка) имеют мягкую тень цвета `--glow`, `box-shadow: 0 0 40px --glow`.
- **Градиентная «сетка-туман»:** на фоне hero и разделах — очень мягкие радиальные размытые пятна cyan/blue (радиал-градиенты, opacity 0.15–0.25), имитирующие боке из референса.
- **Зерно/grain:** едва заметный шум-оверлей (~3% opacity) поверх фона — убирает «пластиковость» и добавляет премиальности.
- **Hover-микроанимации:** плавное свечение и лёгкий подъём (translateY -4px), 200–250ms, ease-out.
- Никаких резких неоново-кислотных ходов — свечение мягкое, атмосферное, «дорогое».

### 2.6 Компоненты

**Кнопка primary:** заливка градиентом `--accent-cyan → --accent-blue`, текст `#05070F` (тёмный, для контраста), скругление 12px, свечение `--glow`. При наведении — усиление свечения.

**Кнопка secondary:** прозрачная, граница `--stroke`, текст `--text-primary`, при наведении — граница и текст `--accent-cyan`.

**Карточка услуги:** glass-стиль, вверху mono-лейбл (категория), крупная цена mono-шрифтом, заголовок, короткое описание, список из 3–4 пунктов с cyan-галочками, кнопка внизу. Одна карточка (пакет) — «выделенная»: чуть ярче свечение + бейдж «Выгодно».

**Бейдж:** маленький, mono, uppercase, граница cyan, полупрозрачный фон.

**Счётчик статистики:** крупное mono-число + мелкая подпись `--text-secondary`.

## 4. Вау-эффект при входе (детально)

Главная задача первого экрана — за 2 секунды вызвать «ого».

**Сигнатурный элемент — «Цифровая радужка»:**
- В hero (по центру или в правой половине) — крупная круглая апертура: 3–5 концентрических колец разной толщины.
- По кольцам бегут мелкие цифры/бинарный код (0 и 1) и точки-«данные» — как в референсе. Внутри — тёмный «зрачок» с мягким cyan-свечением.
- Радужка **медленно вращается** (разные кольца — с разной скоростью) и **реагирует на курсор**: зрачок/блик слегка смещается в сторону движения мыши (эффект «глаз следит за тобой»). На тач-устройствах — реагирует на скролл/наклон или просто мягко «дышит».
- Вокруг — размытые боке-блики (синие + один тёплый янтарный) с лёгким параллаксом.

**Оркестрованная анимация загрузки (page-load, один раз):**
1. Фон проявляется из чёрного (300ms).
2. Радужка «включается» из центра наружу — кольца разворачиваются по очереди со свечением (600–900ms).
3. H1 появляется построчным раскрытием (mask-reveal снизу вверх), слова с задержкой 60ms.
4. Подзаголовок и кнопки — плавный fade-up.
5. Мини-статистика справа — числа быстро «прокручиваются» до финального значения (count-up).

Тайминги мягкие, ease-out. Всё вместе ~1.5 сек, не дольше.

**Атмосфера при скролле:** секции появляются reveal-эффектом снизу с лёгким fade; фоновые боке-пятна медленно двигаются параллаксом. Без перегруза — движение должно ощущаться дорого, а не «дёргано».

**Обязательно:** уважать `prefers-reduced-motion` — при включённом режиме отключаем вращение и параллакс, оставляем статичную радужку и простые fade.

## 5. Адаптивность и качество

- Полностью адаптивно до мобильного (320px+). На мобильном радужка меньше, но остаётся вау-акцентом над заголовком.
- Карточки услуг: 2×2 → 1 колонка на мобильном.
- Видимый фокус для клавиатуры на всех интерактивных элементах.
- Контраст текста к фону не ниже AA.
- Шрифты подключить через Google Fonts (Space Grotesk, Inter, JetBrains Mono).

---

## 6. Что НЕ делать

- Не использовать светлый фон — только тёмная «космическая» база.
- Не заливать сайт янтарным/неоном — свечение мягкое и редкое.
- Не добавлять фейковые логотипы известных компаний и «отзывы» с именами реальных людей.
- Не перегружать анимацией: один сильный вау-элемент (радужка) + сдержанные reveal'ы.
- Не менять цены: сайт — от 40 000 ₽, сценарии — 6 900 ₽, монтаж — 6 900 ₽, пакет — 13 800 ₽.
